A photonic Ising machine encodes a graph's vertices as ±1 spins, then lets a noisy, annealed loop relax toward the lowest-energy configuration — the maximum cut. Cut edges light up as it converges. This runs the same dynamics in your browser, then cleans up the result with a classical output layer.

The output layer. The raw machine usually stalls a few percent short. Apply output layer runs a greedy local search on its output — flipping the single most helpful spin until none helps — which reliably closes the gap. That cheap classical cleanup is the contribution here: it acts only on the machine's output, so it needs no model of the optics.
